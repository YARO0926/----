변수를 꺼내는 법
1.변수를 선언하는 방법
변수를 생성하는 것을 의미한다.
2.변수에 값을 할당하는 방법
변수에 값을 넣는 것을 의히한다.
3.변수를 참조하는 방법
변수에서 값을 꺼내는 것을 의미한다.

파이썬에서 변수 이름을 적어주면 그 자체로 사용하겠다는 선언을 하는 것이다.
여기에 =을 입력하고 값을 입력하면 이것은 값을 변수에 할당하는 것이 된다.

변수->변수에 저장된 값을 출력한다.
변수+변수->변수에 저장된 값으로 연산한다.
print(변수)->변수에 저장된 값을 출력한다.

2-4강

format( )함수:문자열이 가지고 있는 함수
중괄호{}를 포함한 문자열 뒤에 마침표를 찍고 format( ) 함수를 사용하는데,중괄호의 개수와 format 함수 괄호 안
매개변수의 개수는 반드시 같아야한다.

IndexError 예외:{}기호의 개수가 format( ) 개수보다 많으면 발생

{:d}:int 자료형의 정수를 출력하겠다고 직접적으로 지정하는것
매개변수로 정수만 올 수 있다
{:=+d}처럼 앞에 기호를 붙이면 양수와 음수 기호를 표현할 수 있다.

{:@d}:@칸을 빈칸으로 잡고 뒤에서부터 채움

{:f}:float 자료형 출력을 강제로 지정할 때 사용

{:g}:의미없는 0을 제거한 후 출력하고싶을 때 사용


upper( ):문자열의 알파벳을 대문자로 변형
lower( ):문자열의 알파벳을 소무자로 변형


strip( )함수:문자열 양옆의 공백을 제거
istrip( ):왼쪽 공백 제거
rstrip( ):오른쪽 공백 제거


문자열의 구성 파악하기:is@@( )
isalnum( ):문자열이 알파벳 또는 숫자로만 구성되어 있는지 확인
islpha:( ):문자열이 알파벳으로만 구서외어있는 지 확인
isdentifier( ):문자열이 식별자로 사용할 수 있는 것인지 확인
isdecimal( ):문자열이 정수 형태인지 확인
isdigit( ):문자열이 숫자로 인식될 수 있는 것인지 확인
isspace( ):문자열이 공백으로만 구성되어 있는지 확인
islower( ):문자열이 소문자로만 구성되어 있는지 확인
isupper( ):문자열이 대문자로만 구성되어 있는지 확인

isalnum( )은 문자열이 알파벳 또는 숫자로만 구성되어 있는지 확인하는 함수이므로 "Train10"은 True(맞다)이고
isdigit( )는 문자열이 숫자로 인식될 수 있는 것인지 확인하는 함수이므로 "10"도 True(맞다)이다

find( ):왼쪽부터 찾아서 처음 등장하는 위치를 찾음
rfind( ):오른쪽부터 찾아서 처음 등장하는 위치를 찾음

in 연산자:출력은 True 또는 False로 나옴


split( ):문자열을 특정한 문자로 자를 때 사용

f문자열:문자열 앞에 f를 붙여서 만듬
