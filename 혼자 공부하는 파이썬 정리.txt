2강


자료:프로그램이 처리할 수 있는 모든 것

자료형:자료의 형식

문자열:영어-string

\t:탭

\\:역슬래시

\n:줄바꿈

\:코드를 쉽게 보려고 줄바꿈 할 때

인덱스( [  ] ):@번째 위치한 값(0부터 셈) 처음,마지막 까지 하려면 생략 가능(ex:[ :3])

인덱싱:[ ] 연산자를 이용해 문자열의 특정 위치에 있는 문자를 참조하는 것

슬라이싱: [:] 연산자를 이용해 문자열의 일부분을 추출하는 것

len():문자열에 들어있는 문자의 개수를 세어줌


2-2강

int:정수
float:부동소수점(실수)

//:정수 나누기(소수점 떼기)(ex:3//2>>>1)

%:나머지를 구함

**:제곱(ex:2**3 = 8)

계산 시 같은 우선순위를 가진 연산자는 왼쪽에서 오른쪽 순서로 계산함
단,먼저 계산하고싶은 부분은 괄호로 감싸서 계산


2-3강

변수:값을 저장할 때 사용하는 식별자

pi라는 이름의 상자(저장공간)을 만든 후 pi 상자 내부에 값을 넣어 놨다가 필요할 때 이를 호출하여 사용
이때, pi를 변수라고 함

변수를 꺼내는 법
1.변수를 선언하는 방법
변수를 생성하는 것을 의미한다.
2.변수에 값을 할당하는 방법
변수에 값을 넣는 것을 의히한다.
3.변수를 참조하는 방법
변수에서 값을 꺼내는 것을 의미한다.

파이썬에서 변수 이름을 적어주면 그 자체로 사용하겠다는 선언을 하는 것이다.
여기에 =을 입력하고 값을 입력하면 이것은 값을 변수에 할당하는 것이 된다.
